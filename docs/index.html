Asistente IA — Prompts y generación

Modo de generación:
Sugerir campos
Mínimo (README + main)
Completo (README, main, requirements, details)
Completo + Extras (autorun, LICENSE)
Presets: escoge una plantilla rápida para controlar el alcance de la generación. Las respuestas IA deben ser JSON compacto con estructura estricta (ver especificación abajo).
Especificación de prompt (clic para ver)
La IA debe devolver un único bloque JSON con dos campos: meta y files. 
{
  "meta": {
    "empresa":"TechNova",
    "nombre":"weatherly",
    "version":"1.0",
    "titulo":"Weatherly - App Clima",
    "colores": { "primario":"#0b6ef6", "secundario":"#20c997", "fondo":"#f6fbff" }
  },
  "files":[
    {"path":"{folder}/README.md","content":"..."},
    {"path":"{folder}/weatherly.py","content":"#!/usr/bin/env python..."},
    {"path":"{folder}/lib/requirements.txt","content":"flask\nrequests"},
    {"path":"{folder}/details.xml","content":"..."},
    {"path":"{folder}/autorun","content":"#!/bin/sh..."},
    {"path":"{folder}/LICENSE","content":"MIT..."}
  ]
}
